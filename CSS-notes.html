Common ways of writing CSS

  1. In-line CSS
  2. Internal CSS
  3. external CSS

In-line CSS

To style html inline, we simply use the style attribute on any of our html tags e.g.,
<h1 style="color:blue">heading is now blue</h1>

In line styles should never be used!

Internal CSS

To use internal styling, we add a style tag within the head tag e.g.,
<head>
 <style>
  color: blue;
 </style>
</head>

This will then apply a blue color to all h1 tags within our html. This approach seperates our concernes a bit and is a reccommened approach over In-line CSS but still not the best.

External CSS

The preffered approach is to create an external style sheet. We do this by creating a new file called say style.css.

Once we have created our stylesheet, we then need to link the css file to our html doc, we do this in the head tag. We do this with a link tag.

<head>
 <link href="./style.css" rel="stylesheet" />
</head>

Now this style sheet will be applied to our html document.

CSS properties

We always have a selector which determines where to apply the style e.g., if we have h1 {...} this will select all h1 tags and apply this styling to them.

Some other properties we can apply are

  • font-size: 26px; ←Sets the font size to 26px
  • font-family: sans-serif; ←Sets the font family
  • text-transform: uppercase; ←Allows us to transform the text in certain ways such as uppercase.
  • font-style: italic; ←Sets the font to italic, we could also say bold etc.

There is also a line height property which specifies the line height in relation to the size of the text, 1 = same size as text, 1.5 = 150% of text height.

The strong, em and a tags all inherit from a p tag which means that when we apply styling to a p tag, it will automatically be applied to the given tags, it will not override the html functionality however, e.g., a strong tag will always remain strong regardless of CSS.

Another CSS property is the text align which aligns the text on the vertical axis, e.g., left, center right. We would say
 text-align: centre;

Combining selectors

We have just applied font-family to the h1, h2, h3, h4, p and li tags. We can do this in a much better way, we do this by selecting many elements at once.

h1, h2, h3, h4, p, li {
 font-family: sans-serif;
}

This is called a list selector and selects all instances of the list.

Descendent selector

We can sytle an element which is within another element e.g.,
<footer>
 <p>This will be styled</p>
</footer>

The above will be styled below.

footer p {
 font-size: 16px;
}

So all p tags who are children of footer tags will be styled accordingly. This is not overly robust as its dependent on the html structure. This is not scalable.

Using Classes and ID for styling

We can set the id attribute to html tags then set styles based on that. e.g.,

<p id="author">James Jenkins</p>

We then select id's in CSS by using the # symbol e.g.,

#author {
 font-style: italic;
}

The '#' select id's, the '.' selects classes.

Classes vs Id's

The main difference between a class and an Id is that we cannot repeat an id, e.g., if we have an id of 'copywrite', we can only have one of those, if that were say a class, we could have many of them.

Setting class can be done like below.

.related-author {
 font-size: 18px;
 font-weight: bold;
}

Generally in the real world, we never use id's, we only use classes as this allows us to scale the page easily.

Something to note is that currently, some of out p tags are being selected up to three times. We will revisit this later.

Colors

In CSS there are a ref diferent ways we can represent colors, the two main methods are rgb or hex.

To use rgb notation we use say rgb(0, 255, 255), we can also use reba which allows us to specify the transperency of color, 0 fully transparent, 1 fully opaque. rgba(0, 255, 255, 0.3)

to use hex colors we just specify #xxxxxx, rgb in hexidecimal #00ffff this is the same as rgb(0, 255, 255).

When we have all the numbers exactly the same we get a black, white or gray color.

We specify the color of an element using the color property. e.g.,

h1 {
 color: #1098AD;
}

Borders

Borders are another property which we can style with CSS, it is called a short hand property as we can define many things at once. Generally it goes border: <width> <line fill> <color>

So border: 5px, solid, #1098AD; Would create a 5px solid border of color #1098AD. Other styles we can use are dotted, dashed etc.,

There are special variants such as border-top or border-bottom which will just style the top or bottom.
